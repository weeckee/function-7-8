# Банковское приложение на Go

## Основные возможности
- Создание банковского счета  
- Пополнение и снятие средств  
- Переводы между счетами  
-  Просмотр баланса и выписки  
-  Полная история транзакций  

##  Техническая реализация
**Язык**: Go  
**Архитектура**: 2 интерфейса (AccountService, Storage)  
**Ошибки**: Кастомные ошибки валидации  
**Хранение**: In-memory (данные сохраняются до перезапуска)  

##  Быстрый старт
```bash

go run main.go
